Welcome to Stormstats!

Stormstats' purpose is to present hockey statistics from public data sources using open-source (and mostly free) technology. The hope is to provide a framework for others to contribute their sports statistics and data science ideas.

Currently, the site is designed around the Carolina Hurricanes using standard statistics. Future work will incorporate the entire league along with advanced, descriptive statistics and predictive data mining features. The application source code and project progress can be found here.

The following features are currently available:
  • Roster: Details individual player attributes.
  • Schedule: Details the current season's games.
  • Skater Stats: Details collective stats for non-goalie players.
  • Goalie Stats: Details collective stats for goalies.
  • Skater Game Stats:
    • By Game: Details collective game stats for non-goalie players.
    • By Player: Details individual season stats for non-goalie players.
  • Goalie Game Stats:
    • By Game: Details collective game stats for goalies.
    • By Player: Details individual season stats for goalies.
  • About: Details more information.
